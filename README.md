## Local Setup

### Backend

```bash
# Navigate to the backend directory
cd backend

# Install dependencies
npm install

# Run the backend server using nodemon
nodemon index.js
```

## Frontend
```bash
# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm start
```
